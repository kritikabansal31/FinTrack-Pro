# FinTrack Pro

FinTrack Pro is a web-based finance management app that helps users record income and expenses, categorize spending, and view clear financial summaries. It uses JavaScript arrays, objects, ES6+ features, higher-order functions, recursion for future growth prediction, and closures to manage data securely.

## Project Features

- Add income and expense details
- Categorize expenses
- Calculate total income, expenses, and balance
- Show category-wise spending
- Future Growth Predictor using recursion
